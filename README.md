# Seq.App.Discord

An app for Seq (http://getseq.net) that forwards messages to Discord through webhooks.

## Changes

### 0.1.1

- initial revision

## Building NuGet Package

From solution root, run:

- msbuild
- nuget pack ./Seq.App.Discord/Seq.App.Discord.nuspec
